# Scripts Directory

This directory contains Python scripts for data processing and post-simulation analysis.

## What This Contains

Python scripts that enable users to:

- Process raw PhysiBoSS simulation outputs
- Analyze simulation results and experimental data
- Generate visualizations and plots
- Perform statistical analysis on drug response data

## Prerequisites

Python with common scientific packages (numpy, pandas, matplotlib, etc.)
