# Information about each DEAP JSON file run 

## 2p_pressure_control
Contains only 2 params for the pressure Hill transfer function:
1. pressure half max
2. pressure Hill index

## Single-drug experiments
For all of these experiments, we usually have the parameters for:

A. The drug response transfer function
1. drug X half max
2. drug X Hill index
3. drug X permeability

B. The prosurvival-growth rate Hill transfer function
Mapping the pro-survival BM signals to the growth rate
